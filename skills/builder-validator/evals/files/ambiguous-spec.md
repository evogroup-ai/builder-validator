# Employee Leave Management System

## 1. Purpose

System for managing employee leave requests in a mid-size company.

## 2. Leave Types

Employees can request: annual leave, sick leave, personal leave, and other types as needed.

## 3. Request Process

An employee submits a leave request. The manager reviews it. If approved, the leave is recorded.

## 4. Leave Balance

Each employee has a leave balance. The balance is updated when leave is taken.

## 5. Notifications

Relevant parties should be notified about leave requests.

## 6. Reporting

Management needs reports on leave usage.

## 7. Constraints

- Leave requests should follow company policy
- The system should handle edge cases appropriately
- Access should be role-based
